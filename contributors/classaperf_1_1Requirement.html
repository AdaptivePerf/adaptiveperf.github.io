Detailed Description

A class describing a requirement of a profiler that needs to be satisfied before the profiler is used.

◆ check()

bool aperf::Requirement::check ( )
inline

Determines whether the requirement is satisfied.

On the first call, the check is performed and its result is cached. On all subsequent calls, the cached result is returned immediately, regardless of how many objects of a given Requirement-derived class are constructed.
