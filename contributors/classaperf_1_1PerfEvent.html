aperf::PerfEvent::PerfEvent ( )

Constructs a PerfEvent object corresponding to thread tree profiling.

Thread tree profiling traces all system calls relevant to spawning new threads/processes and exiting from them so that a thread/process tree can be created for later analysis.

Parameters
