virtual bool aperf::InitClient::get_profile_start_tstamp ( unsigned long long * tstamp)
pure virtual

Gets the Unix timestamp of the start of profiling and saves it to the variable referenced by tstamp if tstamp is not null.

Returns false if profiling hasn't started yet, true otherwise.

The value referenced by tstamp is unchanged if false is returned or tstamp is null.

Parameters
tstampA pointer to the variable where the profiling start timestamp should be stored. It can be null.
