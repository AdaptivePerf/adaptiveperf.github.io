Detailed Description

A class describing the configuration of CPU cores for profiling.

Specifically, CPUConfig describes what cores should be used for post-processing + profiling, what cores should be used for running the command, what cores should be used for both, and what cores should not be used at all.

Parameters
maskA CPU mask string, where the i-th character defines the purpose of the i-th core as follows: ' ' means "not used", 'p' means "used for post-processing and profilers", 'c' means "used for the profiled command", and 'b' means "used for both the profiled command and post-processing + profilers".
