Starts the profiler.

Parameters
pidThe PID of a process the profiler should be attached to. This may be left unused by classes deriving from Profiler.
connection_instrsadaptiveperf-server connection instructions, sent by adaptiveperf-server during the initial setup phase.
result_outThe path to the "out" directory of results of the current profiling session.
result_processedThe path to the "processed" directory of results of the current profiling session.
capture_immediatelyIndicates whether event capturing should become immediately after starting the profiler. If set to false, the call to start() must be followed by the call to resume() at some point.
